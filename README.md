# Проєкти на C з багатопотоковістю та синхронізацією

Цей репозиторій містить два завдання на **C** з використанням **pthread** для багатопотокових обчислень і синхронізації.

---

## Завдання 1: Обчислення інтегралу методом середніх прямокутників

### Опис
Програма обчислює інтеграл:

\[
\int_0^2 (4 - x^2) dx
\]

методом середніх прямокутників з розбиттям на `N = 1,000,000` підінтервалів.  
Обчислення паралелізується на `THREAD_COUNT = 4` потоках.  
Глобальна змінна `global_result` захищається **м’ютексом**, щоб уникнути конфліктів при одночасному доступі потоків.
